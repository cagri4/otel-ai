**1. Create Server Actions** at `src/app/(dashboard)/employees/actions.ts`:

`'use server'` module with two actions:

**toggleAgent(formData: FormData):**
- Extract agentId and current is_enabled from formData
- Auth check: `const supabase = await createClient(); const { data: { user } } = await supabase.auth.getUser();` + guard
- Update: `supabase.from('agents').update({ is_enabled: !currentEnabled }).eq('id', agentId)` (RLS scopes to hotel)
- Call `revalidatePath('/employees')`

**updateAgentConfig(formData: FormData):**
- Extract agentId, tone (string), custom_instructions (string) from formData
- Build behavior_config JSONB: `{ tone, custom_instructions }`
- Update: `supabase.from('agents').update({ behavior_config: config }).eq('id', agentId)` (RLS scoped)
- Call `revalidatePath('/employees')`

Use the established patterns: `(supabase as unknown as SupabaseClient)` cast for update if needed, `ReturnType<typeof supabase.from>` cast. Auth pattern matches settings/actions.ts.
