  <action>
Create `src/app/api/whatsapp/webhook/route.ts` implementing the full WhatsApp message pipeline:

```
export const runtime = 'nodejs';
export const dynamic = 'force-dynamic';
```

Export `async function POST(req: Request): Promise<Response>`:

1. **Parse body** — Twilio sends `application/x-www-form-urlencoded`:
   ```
   const rawBody = await req.text();
   const params = Object.fromEntries(new URLSearchParams(rawBody));
   ```

2. **Validate Twilio signature** — import `twilio` default, use `twilio.validateRequest()`:
   ```
   const signature = req.headers.get('x-twilio-signature') ?? '';
   const webhookUrl = `${process.env.NEXT_PUBLIC_APP_URL}/api/whatsapp/webhook`;
   const isValid = twilio.validateRequest(
     process.env.TWILIO_AUTH_TOKEN!,
     signature, webhookUrl, params
   );
   if (!isValid) return new Response('Forbidden', { status: 403 });
   ```

3. **Extract fields**: `From` (guest phone), `Body` (message), `To` (Twilio number).

4. **Resolve hotel**: Call `resolveHotelFromNumber(params.To)`. Return 404 if null.

5. **Per-hotel rate limit**: Call `checkHotelRateLimit(hotelId)`. Return 429 if not success.

6. **Sanitize input**: Call `sanitizeGuestInput(params.Body)`.

7. **Derive conversation ID**: `wa_${hotelId}_${guestPhone}` where guestPhone = `From.replace('whatsapp:', '')`.

8. **Invoke agent** (non-streaming — WhatsApp needs full response):
   ```
   const response = await invokeAgent({
     role: AgentRole.FRONT_DESK,
     userMessage: sanitizedBody,
     conversationId,
     hotelId,
     guestIdentifier: guestPhone,
     // No onToken — WhatsApp needs complete message
   });
   ```

9. **Send reply**: Call `sendWhatsAppReply({ from: params.To, to: params.From, body: response })`.

10. **Return 200** to Twilio: `return new Response('', { status: 200 });`

Wrap the entire handler (steps 3-9) in try/catch. On error, log the error but still return 200 to Twilio to prevent retries.

IMPORTANT: Do NOT use `req.json()` — Twilio sends form-urlencoded. Do NOT use SSE/streaming — WhatsApp expects a single complete message.
