Create `supabase/migrations/0008_housekeeping.sql` with three tables:

**1. room_housekeeping_status** (one status row per room):
- `id` UUID PK DEFAULT gen_random_uuid()
- `hotel_id` UUID NOT NULL FK hotels(id) ON DELETE CASCADE
- `room_id` UUID NOT NULL FK rooms(id) ON DELETE CASCADE
- `status` TEXT NOT NULL DEFAULT 'dirty' CHECK (status IN ('clean', 'dirty', 'inspected', 'out_of_order'))
- `notes` TEXT nullable
- `updated_by` TEXT (values: 'agent', 'cron', 'owner')
- `updated_at` TIMESTAMPTZ NOT NULL DEFAULT NOW()
- UNIQUE(hotel_id, room_id)
- Enable RLS. Policy for authenticated SELECT using hotel_id from JWT. Service role policy for ALL with CHECK(true).
- Index on (hotel_id).

**2. housekeeping_queue** (daily priority queue, one row per room per date):
- `id` UUID PK DEFAULT gen_random_uuid()
- `hotel_id` UUID NOT NULL FK hotels(id) ON DELETE CASCADE
- `room_id` UUID NOT NULL FK rooms(id) ON DELETE CASCADE
- `queue_date` DATE NOT NULL
- `priority` INTEGER NOT NULL (1=checkout today highest, 2=checkin today, 3=checkin tomorrow)
- `reason` TEXT NOT NULL (e.g. 'checkout_today', 'checkin_today', 'checkin_tomorrow')
- `assigned_to` TEXT nullable (staff name or email after assignment)
- `assigned_at` TIMESTAMPTZ nullable
- `completed_at` TIMESTAMPTZ nullable
- `created_at` TIMESTAMPTZ NOT NULL DEFAULT NOW()
- UNIQUE(hotel_id, room_id, queue_date) for idempotency
- Enable RLS. Same policies as room_housekeeping_status.
- Index on (hotel_id, queue_date).

**3. housekeeping_staff** (staff directory for task assignment):
- `id` UUID PK DEFAULT gen_random_uuid()
- `hotel_id` UUID NOT NULL FK hotels(id) ON DELETE CASCADE
- `name` TEXT NOT NULL
- `email` TEXT NOT NULL
- `phone` TEXT nullable
- `is_active` BOOLEAN NOT NULL DEFAULT TRUE
- `created_at` TIMESTAMPTZ NOT NULL DEFAULT NOW()
- Enable RLS. Authenticated SELECT by hotel_id JWT. Service role ALL.
- UNIQUE(hotel_id, email)
